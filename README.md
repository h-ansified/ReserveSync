# ReserveSync
Restaurant Reservation Management Made Simple
Streamline your restaurant operations with our multi-tenant reservation system. Manage tables, send automated emails, and provide seamless booking experiences for your customers.
Everything You Need to Manage Reservations
Powerful features designed to simplify restaurant management and enhance customer experience

Multi-Tenant Architecture
Complete data isolation for each restaurant with dedicated Google Sheets and Gmail integration

Automated Email Confirmations
Send branded confirmation emails automatically using your restaurant's Gmail account

Role-Based Access Control
SuperAdmins manage all restaurants while owners access only their specific data

Smart Table Management
Organize tables with capacity tracking, status updates, and availability management

Public API Integration
Connect your website with secure API keys for seamless reservation bookings

Custom Email Templates
Create personalized email templates for confirmations, cancellations, and reminders
